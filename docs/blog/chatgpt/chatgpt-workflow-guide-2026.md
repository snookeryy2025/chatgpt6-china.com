## 一、写作场景：从憋稿到秒产

### 1. 写专业邮件（3种高频场景）

**场景A：求职/跳槽邮件**

```
【提示词模板】
帮我写一封求职邮件：
- 应聘岗位：[具体岗位名称]
- 我的优势：[3条核心亮点，用一句话概括每条]
- 目标公司：[公司名/行业]
- 语气要求：专业但不生硬，有热情但不夸张

参考格式：
1. 开头：简要说明从哪里看到这个职位
2. 主体：为什么我适合这个岗位（用数据说话）
3. 结尾：期待进一步沟通，注明联系方式
```

**场景B：跨部门沟通邮件**

```
【提示词模板】
帮我写一封跨部门协作邮件：
- 我的部门：[你的部门]
- 对方部门：[协作部门]
- 请求事项：[需要对方做什么]
- 截止时间：[具体日期]
- 背景：[为什么需要他们配合]

语气要求：有理有据，尊重对方立场，强调双赢
```

**场景C：向上级汇报进展邮件**

```
【提示词模板】
写一封项目周报邮件：
- 项目名称：[项目名]
- 本周完成：[3-5个具体事项，越具体越好]
- 下周计划：[3个左右的目标]
- 当前风险：[如果有]
- 需要支持：[需要领导/同事提供什么]

结构要求：结论先行，数据支撑，控制在300字以内
```

### 2. 做PPT大纲与内容填充

ChatGPT可以帮你快速生成PPT的结构和内容框架：

```
【提示词模板】
为一个 [主题] 的PPT生成大纲，要求：
- 共 [N] 页
- 每页包含：标题、3个核心要点、1个数据/案例支撑
- 受众是 [目标受众]
- 风格：[商务/学术/活泼等]

输出格式：用Markdown格式，便于直接复制到PPT工具
```

::: tip 💡 进阶技巧
生成PPT内容后，可以用 **Gamma.app** 或 **Beautiful.ai** 等工具，粘贴内容直接生成精美PPT，效率再翻倍。
:::

## 二、数据处理：告别Excel恐惧

### 3. Excel公式生成与解释

很多人面对复杂的Excel公式就头大，ChatGPT可以帮你写公式并解释原理：

```
【提示词模板】
我有一张Excel表，结构如下：
- A列：[第一列的内容描述]
- B列：[第二列的内容描述]
- C列：[第三列的内容描述]

我需要：
[描述你想要的结果，比如：计算A列>100且B列包含"北京"的行数]

请给出Excel公式，并解释每个部分的作用。
```

### 4. 数据分析与解读

```
【提示词模板】

1. 找出3个最重要的发现
2. 用非专业人士能听懂的语言解释
3. 给出2-3个可以采取的行动建议
```

### 5. SQL查询语句

```
【提示词模板】
```

## 七、提示词进阶：让AI更懂你

### 16. 结构化提问公式

想让ChatGPT的回答质量提升一个档次？记住这个公式：

```
角色 + 任务 + 背景 + 约束 + 格式 = 高质量回答

示例：
[你是] 一位资深HR [角色]
[请帮我] 优化这份简历 [任务]
[背景是] 我正在投递 [某行业/某岗位] [背景]
[要求是] 控制在1页以内，突出 [某方面优势] [约束]
[输出格式] 先给整体评价，再逐条修改建议 [格式]
```

### 17. 多轮对话迭代优化

不要期待一步到位，**好的回答来自多轮迭代**：

```
第一轮：描述需求，获取初稿
第二轮：指出需要调整的地方
第三轮：进一步优化
...
直到满意为止
```

### 18. 让ChatGPT扮演专家角色

```
【提示词模板】
请以 [专家身份] 的视角，审视/评估/建议 [你的内容]：
[粘贴你的内容]

扮演的专家需要具备：
- [专业背景描述]
- [行业经验描述]
- [特有的分析框架或标准]

请给出专业、中肯、有建设性的反馈。
```

## 八、安全与边界：高效的前提是安全

::: warning ⚠️ 重要提醒
- **不要输入**：银行卡密码、身份证号、公司核心机密、未经授权的用户数据
- **验证关键信息**：法律、医疗、金融等专业领域的建议仅供参考，需咨询专业人士
- **保护隐私**：提问时用"假设""比如"替代真实敏感信息
:::
