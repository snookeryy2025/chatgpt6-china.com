## 三、最大化免费额度的使用技巧

### 1. 精简 Prompt，节省 Tokens

**原则：** 每个字符都消耗 tokens，优化 Prompt 可以节省额度。

❌ **浪费 tokens 的做法：**
```
你好 ChatGPT，我想问你一个问题，
我最近在学习编程，遇到了一些困难，
不知道该怎么办，你能不能帮我写一个
Python 程序，用来计算1到100的和？
谢谢你！
```
消耗约 150 tokens

✅ **节省 tokens 的做法：**
```
用 Python 计算1到100的和，提供代码
```
消耗约 30 tokens

**节省技巧：**
- 去掉客套话
- 直接说明需求
- 使用简洁的表达
- 避免重复描述

---

### 2. 一次性问完整，避免多次追问

❌ **低效方式：**
```
第1次：什么是机器学习？
第2次：机器学习有哪些应用？
第3次：怎么学习机器学习？
第4次：推荐一些入门书籍
```
总消耗：约 1000 tokens

✅ **高效方式：**
```
请介绍机器学习，包括：
1. 定义和原理
2. 主要应用场景
3. 学习路径建议
4. 入门书籍推荐
```
总消耗：约 400 tokens

**节省 60% 的 tokens！**

---

### 3. 利用代码块和格式化输出

让 ChatGPT 输出结构化内容，便于后续使用，减少重复提问。

**示例：**
```
请以 JSON 格式输出一份学习计划：
- 主题
- 每周学习内容
- 预计时间
- 推荐资源

这样我可以直接保存，不用再次提问
```

---

### 4. 使用"继续"而非重新提问

当回答被截断时：

❌ **不要重新发送完整问题**

✅ **直接发送"继续"或"请继续"**

---

### 5. 批量处理任务

把多个相似任务合并为一次请求。

**示例：**
```
请帮我完成以下3个翻译：
1. [英文句子1]
2. [英文句子2]
3. [英文句子3]

请按序号输出中文翻译
```

---

### 6. 使用模板和复制粘贴

对于重复性任务，准备好模板：

**示例模板：**
```
请将以下文本改写为[风格]：
[文本内容]

要求：[具体要求]
```

每次只需要替换变量部分，节省思考和输入时间。
