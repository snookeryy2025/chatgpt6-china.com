# 3. ChatGPT 是什么？为什么这么火？

**ChatGPT 是 OpenAI 开发的生成式人工智能聊天机器人**，基于大型语言模型（LLM），能够：

* 回答问题
* 写文章、写代码、写文案
* 翻译、总结、解释专业概念
* 生成图像、分析文件
* 进行多轮聊天，甚至语音通话

最新模型 **GPT-5.1（2025年11月发布）** 在速度、逻辑推理、语言理解方面大幅提升，是目前全球最强的 AI 模型之一。

# 4. 为什么国内访问 ChatGPT 困难？（核心原因）

国内用户无法直接访问 ChatGPT 官网，主要有两点：

### ① 网络限制

OpenAI 未对中国开放访问权限，官网会被阻止。

### ② 注册验证门槛

官网注册需要海外手机号，大多数国内用户无法获取。

因此，**国内镜像站成为最现实的替代方案**。
