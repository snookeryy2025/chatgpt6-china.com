## 一、创作搭档：让你的灵感不再枯竭

### 1. 小说/故事创作引擎

ChatGPT最让人惊艳的创作能力之一，就是帮你从0到1构建故事世界：

```
【提示词模板】
我想写一个 [题材，比如：都市悬疑/科幻/言情] 的短篇小说，帮我构建以下内容：

1. 故事大纲（起承转合，每阶段200字左右）
2. 主要人物设定（3个核心角色，包含性格特点、背景故事、人物弧光）
3. [N] 个关键情节点
4. 开头第一句（请给出5个不同风格的版本）

故事基调：[悲壮/轻松/黑暗/温暖等]
目标读者：[年龄段/类型偏好]
篇幅预期：[短篇/中篇/长篇]
```

进阶玩法：让ChatGPT分段生成内容，你来做"调色板"，不断微调风格、情节走向，直到接近你心中的故事。

### 2. 歌词/诗歌创作

```
【提示词模板】
请为我创作一首 [类型：民谣/流行/说唱/古风] 风格的歌词：
- 主题：[你想表达的情感或故事]
- 情绪：[忧伤/热血/治愈/叛逆等]
- 长度：[4段/8段/完整版]
- 风格参考：[某位歌手/某首歌的味道]

如果有押韵要求，请标注韵脚。
```

### 3. 剧本/短视频脚本

短视频时代，一个好脚本就是流量的起点：

```
【提示词模板】
为一个 [X] 秒的 [类型：搞笑/知识干货/情感/种草] 短视频写脚本：

主题：[视频核心内容]
目标平台：[抖音/小红书/B站/视频号]
目标受众：[人群描述]
目标效果：[比如：引发共鸣/促使分享/引导购买]

请给出：
1. 封面文案（3个选项）
2. 前3秒的"钩子"设计
3. 正文脚本（带时间戳）
4. 结尾行动号召（CTA）
5. 配乐/音效建议
```

## 二、艺术搭档：人人都是艺术家

### 4. AI绘图提示词生成器

虽然ChatGPT本身不直接生成图片，但你可以用它来**写Midjourney/DALL-E/Stable Diffusion的提示词**：

```
【提示词模板】
我想画一幅 [风格：赛博朋克/水彩/蒸汽朋克/国风/像素风] 的 [主题] 插画：

画面元素：[列出你想要的元素]
情绪/氛围：[冷峻/温暖/神秘/欢快]
视角：[平视/俯视/仰视/特写/全景]
参考艺术家风格：[如：宫崎骏/莫奈/ASK风/，国内可用的替代工具推荐：]

请生成：
1. 详细的英文提示词（适合Midjourney/DALL-E）
2. 负面提示词（不想出现的元素）
3. 参数建议（比例、风格化程度等）
```

### 5. Logo与品牌视觉设计

```
【提示词模板】
我想设计一个 [行业/产品名称] 的Logo，请给出：

1. 设计理念（3个不同方向）
2. 每个方向的：
   - 核心图形元素描述
   - 配色方案（含色号）
   - 字体建议
   - 应用场景说明
3. 简约版与完整版Logo的草图描述（文字可描述）

品牌调性：[高端/亲民/专业/活泼/复古]
```

## 三、音乐与技术：代码也能写曲子

### 6. 生成可演奏的简谱/五线谱

ChatGPT可以帮你创作简单的旋律，并输出为简谱或描述性的乐谱：

```
【提示词模板】
请为以下描述创作一段音乐：

- 类型：[纯音乐/背景配乐/主题曲片段]
- 情绪：[欢快/忧伤/史诗感/宁静]
- 用途：[短视频配乐/游戏BGM/个人练习]
- 长度：[约30秒/60秒/更长]
- 调式：[C大调/A小调/自由调]
- 速度：[如：120BPM]

请用简谱格式输出主旋律，并标注：
- 拍号
- 调号
- 主要和弦进行
- 可选：简要说明每个乐句的设计意图
```

### 7. 生成MIDI文件描述

对于会使用DAW（数字音乐工作站）的用户：

```
【提示词模板】
请用音乐理论的语言，描述一个适合 [场景] 的 [X] 秒音乐片段：

请给出：
1. 整体结构（引子-主歌-副歌-尾声等）
2. 每段的调式、节拍、速度建议
3. 主旋律的音高走向描述（用文字描述）
4. 和弦进行（标注级数与具体和弦）
5. 配器建议（建议使用的乐器）
```

## 四、游戏开发：一个人也能做游戏

### 8. 桌游/剧本杀剧本设计

```
【提示词模板】

故事背景：[简要背景设定]
游戏时长：[约X小时]
难度：[新手/进阶/硬核]

请输出：
1. 游戏背景故事（300字左右）
2. 人物角色卡（每个角色包含：姓名、背景故事、隐藏任务、胜利条件）
3. 游戏流程（每个环节的操作指引）
4. 主持人的特殊事件卡（3-5张）

### 9. 文字冒险游戏设计

```
【提示词模板】

玩家角色：[1个起始设定]
游戏时长：[短篇（10个选择以内）/中篇/长篇]
结局数量：[N个不同结局]
难度：[轻松/中等/挑战]

请输出：
1. 游戏开场场景
2. 至少 [N] 个分支场景（每个包含：场景描述、选项、结果）
3. 关键道具/属性系统（如有）
4. 各结局的达成条件
```

### 10. 微信小程序/小游戏创意策划

```

目标用户：[年龄段/兴趣特征]
核心玩法：[简要描述]
差异化亮点：[你的创新点]

请给出：
1. 产品概念（1句话）
2. 核心玩法流程图（文字描述）
3. 用户成长体系

本周主题：[比如：职场沟通技巧/春季穿搭/读书分享]

## 七、学习探索：AI打开新世界的大门

### 18. 模拟面试与演讲练习

```
【提示词模板】
我想请你扮演 [面试官/演讲评委/谈判对手]，和我进行 [场景] 练习：

场景详情：
- 面试岗位/演讲主题：[具体内容]
- 我最担心的问题：[如：技术问题/紧张忘词/被追问]
- 练习目标：[如：克服紧张/优化表达逻辑]

请：
1. 先给我一些准备建议
2. 然后开始模拟，给我即兴提问/挑战
3. 最后给我反馈与改进建议
```

### 19. 历史/科学事件"穿越"视角

```
【提示词模板】
请用 [古代人/未来人/外星人/孩子] 的视角，描述 [某个现代科技/日常场景]：

例如："如果一个唐朝人看到智能手机，他会怎么想？"
或者："如果一个2110年的人穿越到现在，会对什么感到震惊？"

要求：
- 语言风格贴合设定角色的认知水平
- 包含3-5个具体的观察细节
- 有趣但有理有据
```

### 20. 哲学辩论与思想实验

```
【提示词模板】
请和我进行一次 [主题：AI伦理/自由意志/存在主义/其他哲学话题] 的苏格拉底式对话：

我的初始观点：[你的想法]
对话目标：[理解更深层的逻辑/挑战自己的认知/寻找答案]

请用不断追问的方式，引导我深入思考，而不是直接给我结论。
```

## 八、进阶玩法：解锁ChatGPT的隐藏技能

### 21. 构建个人AI知识库

把ChatGPT变成你的"第二大脑"：

```
【提示词模板】
我想建立一个关于 [你的专业领域/兴趣爱好] 的知识库。请帮我：

1. 列出这个领域最重要的 [N] 个核心概念
2. 为每个概念给出：定义、理解要点、常见误解、应用场景
3. 推荐3本入门书/5个进阶资源
4. 设计一个学习路径（从入门到精通的时间规划）

我的基础水平：[零基础/有一些了解/已经比较专业]
每周可用于学习的时间：[约X小时]
```

### 22. 多语言翻译与文化适配

```
【提示词模板】
请将以下内容翻译成 [目标语言] 并做文化适配：

原文：[粘贴内容]

请同时给出：
1. 直译版本
2. 文化适配版本（根据目标受众调整表达方式）
3. 翻译/改写说明
4. 目标文化中的禁忌或注意事项
```
