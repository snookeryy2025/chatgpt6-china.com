## 四、安全与隐私保护指南

### 4.1 镜像站选择三大原则

在选择国内镜像站时，务必验证以下要素：

**① 技术安全性**
- ✅ 检查网站是否使用 HTTPS 加密
- ✅ 查看隐私政策是否明确数据处理方式
- ✅ 测试是否要求过度权限（如读取通讯录）

**② 商业合规性**
- ✅ 是否在工信部备案（ICP备案号）
- ✅ 企业资质是否可查（天眼查/企查查）
- ✅ 运营时长（建议选择1年以上平台）

**③ 用户口碑**
- ✅ 在知乎/微博搜索平台名称
- ✅ 加入官方QQ群/Telegram群观察活跃度
- ✅ 查看是否有大量投诉记录

**推荐验证流程：**
```bash
# 1. 检查SSL证书
curl -vI https://目标网站.com

# 2. 查询ICP备案
访问 beian.miit.gov.cn

# 3. 企业资质核验
访问 tianyancha.com
```

### 4.2 数据泄露风险防范

**高危操作清单（禁止在 ChatGPT 中操作）：**
- ❌ 输入完整的身份证号、护照号
- ❌ 粘贴企业内部代码（包含数据库密码）
- ❌ 分享未公开的商业计划书
- ❌ 上传包含人脸的私密照片

**安全实践建议：**
1. **信息脱敏**：在提问前替换敏感字段
   ```
   错误示例：我的密码是 Abc@123456
   正确示例：我的密码格式类似 [大写字母][小写][特殊符号][数字]
   ```

2. **使用独立账号**：工作与个人账号分离

3. **定期清理记忆**：在 ChatGPT 设置中删除对话历史

4. **企业级方案**：使用 [GPT-4o 极速体验站 (chatgpt-blog.com)](https://chatgpt-blog.com) 的企业版，提供数据加密和审计功能
